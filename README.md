# Student-Dropout-Prediction
This project focuses on predicting whether a student is likely to drop out based on academic, social, and personal factors using Machine Learning.

It is built as a resume-level ML project for aspiring Data Analysts / Data Scientists, with clean structure, preprocessing, model training, and evaluation.
